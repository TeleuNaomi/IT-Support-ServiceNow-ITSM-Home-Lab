# IT-Support-ServiceNow-ITSM-Home-Lab
ServiceNow is a cloud-based IT Service Management (ITSM) platform used by organizations to manage help desk operations. It allows IT teams to create, track, resolve, and document support tickets while following structured workflows such as incident management and service requests.

The main objectives of this lab are:

- Setting up a ServiceNow Personal Developer Instance (PDI)
- Managing users and support groups
- Creating and handling IT incident tickets
- Processing tickets following real-world IT technician procedures

1) Installing ServiceNow PDI
- Create an account and sign in at: https://developer.servicenow.com/
- Navigate to Learning > Developer Resources > Explore Resources
You will be asked if you know how to code. Answer according to your skills; it doesn’t affect the DPI. You will be asked about your responsibilities. Choose “IT Admin”.
- You will be directed to your dashboard. The Dashboard contains your credentials.
NB: It is important to send your credentials to your personal mail
-	Click “Start Building” to access the main application used by many companies

2) Creating Company groups and Users
-	Click “All,” then type “ User Administration”, select “Users”
-	Top right, Select “New” and fill in the Blank spaces. Once you are done, you submit
-	To create a Group: Click “All,” then type “ User Administration”, select “Groups”
-	Fill in blanks and submit

3)	Creating an Incident
-	Click “All,” then type “ Incident”, select “New”
Your new employee, James Jamovitch can’t access his email account. A ticket is created and assigned to IT Securities.
-	Once submitted, you will see it appear on your dashboard

  4)	Processing a Ticket like an IT technician
-	You can click on the Incident number. Write all the actions taken, resolution information, and select “ Resolve”
